## üöÄ Local Development

### Prerequisites
- Node.js v18+
- npm v9+

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/RedRiotTank/portfolio.git
   cd portfolio
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development server:
    ```bash
     npm run dev
     ```
4. Open on the web on your browser:
    ```bash
     (http://localhost:4321
     ```
